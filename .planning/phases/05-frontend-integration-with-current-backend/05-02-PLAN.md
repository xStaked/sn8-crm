<interfaces>
Required API shape:
  GET /conversations/:conversationId/messages
    200 [
      {
        id: string,
        conversationId: string,
        direction: 'inbound' | 'outbound',
        body: string | null,
        createdAt: string
      }
    ]

History rules:
  - filter by the same stable conversation id emitted by GET /conversations
  - sort ascending by createdAt so the frontend can render top-to-bottom history
  - reject access to a conversation id that does not map to any rows with HTTP 404
</interfaces>
