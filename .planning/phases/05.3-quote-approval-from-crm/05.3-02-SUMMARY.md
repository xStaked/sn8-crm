- Added CRM-safe `approveDraftFromCrm()` and `requestChangesFromCrm()` entrypoints in `OwnerReviewService` with transport-aware audit text.

## Decisions Made

- Reused `OwnerReviewService` as the only approval engine and added thin CRM entrypoints instead of introducing a second CRM-only approval path.
- Returned the existing `ConversationQuoteReviewDto` after each write action so the frontend can refresh from the same detail contract it already reads.
