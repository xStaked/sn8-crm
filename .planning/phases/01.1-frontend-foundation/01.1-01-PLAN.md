1. Create the Next.js app:
   ```bash
   npx create-next-app@14 apps/web --typescript --tailwind --app --src-dir --import-alias "@/*" --no-git
   ```

2. Add `"apps/web"` to root `package.json` workspaces array (currently `["apps/backend"]`, change to `["apps/backend", "apps/web"]`).

3. Set `apps/web/package.json` name to `"@sn8-wpp-crm/web"`.

4. Initialize shadcn/ui inside `apps/web`:
   ```bash
   cd apps/web && npx shadcn@latest init
   ```
   When prompted (or via flags): Style: Default, Base color: Zinc, CSS variables: Yes.
   This creates `components.json` and updates `tailwind.config.ts` with CSS variable references.

5. Install all required shadcn components:
   ```bash
   cd apps/web && npx shadcn@latest add button input label card separator skeleton avatar tooltip badge
   ```

6. Install SWR:
   ```bash
   cd apps/web && npm install swr
   ```

7. Create `apps/web/.env.local`:
   ```
   NEXT_PUBLIC_API_URL=http://localhost:3001
   ```

8. Ensure `apps/web/next.config.ts` has no conflicting settings. Keep the default Next.js 14 config. No custom rewrites or redirects needed.

CRITICAL: Do NOT run `npx shadcn@latest add dashboard-01` — use it as structural inspiration only. The 3-column layout will be built from scratch per the UI spec.

   ```typescript
   const API_URL = process.env.NEXT_PUBLIC_API_URL ?? 'http://localhost:3001';

   export async function apiFetch(
     path: string,
     options: RequestInit = {},
   ): Promise<Response> {
     return fetch(`${API_URL}${path}`, {
       ...options,
       credentials: 'include',
       headers: {
         'Content-Type': 'application/json',
         ...options.headers,
       },
     });
   }
   ```
